# codificar 

Usamos o CodBlocks para executar o código. Utilizamos a palavra "Olá" para codificar e decodificar para vermos se deu certo.

Coloquei as imagens de exemplo:

Na primeira o usuário escolhe a opção de criptografar uma mensagem:

Mensagem digitada: ola
Chave (deslocamento): 3
Resultado: rod
Isso acontece porque cada letra foi deslocada 3 posições para frente no alfabeto (Cifra de César).

Já na segunda imagem:

O usuário escolhe a opção de descriptografar:

Mensagem digitada: rod
Chave (deslocamento): 3
Resultado: ola
Aqui, o programa faz o processo inverso, deslocando as letras 3 posições para trás.
